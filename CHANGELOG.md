# Changelog

## [Unreleased] - Sprint 10（2026-07-11）

### Added
- 回収袋システム: 掘った資源はその場で袋へ（拠点への運搬往復を廃止）。大画面を開いた瞬間に一括換金「回収 +Nコイン（土×12…）」。小窓には $コイン(+袋) を表示。セーブv3（運搬中の手下は自動移行）
- ショップv1（データ駆動 data/shop/）: 手下を雇う（50〜、×1.5）／つるはし強化（100〜、×2）／地質調査（120〜、×2）。レベル・コスト・効果はセーブに保存
- カメラ改善: 掘削中の手下を追尾（土ブロックと掘るモーションが同じ列に映る）
- 宝箱仕様変更: 常時コイン10枚＋未所持コレクション1種（コンプ後もコイン継続）
- テスト計65件

## [Unreleased] - Sprint 9（2026-07-11）

### Added
- コイン経済: 納品時に資源を自動換金（土1/石2/鉱石5/魔石10）。ショップ機能の土台。部屋の建設費もコイン建てに変更（宿舎25/酒場40/祭壇80）
- 金塊（3%）: 掘ると+25コイン
- 宝箱（2%）: アイテム5種からランダム入手（全所持時は+10コイン）。「宝物」ボタンでコレクション閲覧
- セーブv2移行: 旧セーブの資源在庫を初回ロード時に自動換金
- ストリップ表示を1列ズームに（セル=窓の高さ、手下に目と作業モーション、群れ追尾カメラ）
- 経済テスト7件（計57件）

## [Unreleased] - Sprint 8（2026-07-11）

### Added
- デイリー異変のコア（§5.4）: 日付→FNV-1aシードで全プレイヤー共通の異変が毎日切替。肥沃な地脈（文書+10%）/柔らかい土（掘削+1）/金粉の風（掘るたび金+1）の3種。展開画面のHUDに「今日の異変」を表示。調査書カードPNGと専用ミニ地層は次段階
- デイリーテスト7件（計50件）

### Changed
- ストリップを全幅の帯から**右下タスクバー領域の小窓（320×48）**に変更（時計・トレイを避けて配置、カメラは手下を横方向にも追尾）
- 自動掘削「一列ずつ」: 最浅の未完了行を掘り切ってから次の層へ（竪坑の脇に土柱が残らない）

## [Unreleased] - Sprint 4〜6（夜間自律セッション 2026-07-11）

### Added
- タスクバー型ストリップ（48px、高さ48/120/180/240切替）。カメラが掘削前線を自動追尾、資源の半透明表示、未読文書は右端で点滅

- 書庫ダイアログ拡大（860×500、本文17px）。ストリップ内でクリップされて手記が読めない問題を解消

- 深度による暗がり表現、手下の作業ボブアニメ・個体色
- 新規ゲームは「自動:下へ掘る」で開始（放置ゲーは初手から放置できるべき）

- 4層目「古代遺構」（深度31〜50、硬度9、魔石産出）＋文書5編追加（採掘許可状、壁の刻み書き、発掘日誌、祭文、手記・四。計13編）
