## 1. 東国の仲間 — 月夜 円(つくよ まどか)

- **名前**: 月夜 円(つくよ まどか)。「円」は仲間4人の小さなエピソードアイテムに共通して刻まれる「欠けた円(アナスタシスの象徴)」と、意味の上で静かに繋がる
- **一族**: **月夜家**。藩主直属の武士であり、**東国で最強と謳われた一族**。歴代当主は藩主の剣術指南役 兼 親衛隊長を務めてきた
- **技の継承方法**: 口伝ではなく、**幼少期からの過酷な修行に耐え抜くことでのみ剣技を会得する**という厳格な方式。この継承方法が、円が記憶を失ってもなお剣技だけは体に染み付いている理由になる
- **家紋**: 円形の意匠
- **属性**: 闇（空間転移で距離を詰めて敵を切るアタッカー）
- **表向きの物語**: 藩主暗殺の濡れ衣を着せられ、記憶を失ったまま追われる身になっている
- **裏の真実（通常レベル）**: 藩主は円の正体に気づき、守ろうとして守人に消された。円ごと消すため濡れ衣を着せられた

### 1.1 円の特別な真実【物語終盤限定・最重要ネタバレ】

円は他の3人（ヴァルド・司馬燿・サユ）とは根本的に異なる存在として設計する。

- 千年前、円はもともと5人の生贄の一人として選ばれていたが、**双子の姉が身代わりとなり**、実際に儀式で失われたのは姉の方だった
- 円自身は、**時間が止まる特殊な場所**へ連れて行かれ、**999年間眠り続けた**（この場所は大穴とは別の場所にある）
- 999年という眠りの長さそのものが、円の記憶喪失の原因
- 目覚めた場所は東国の国内ではなかった。目覚めた場所の近くで倒れていたところを、旅の途中のソティリス(たち)に拾われる（これが物語冒頭の出会いになる）
- 東国への帰還（円の外伝、4人の中で最後に実施）にて、**血縁も、親しいと思っていた人々も、実はすべて千年前の関係者の子孫でしかない**という事実に直面する
- 一族の記録を辿ると、**円自身が月夜家の初代当主だった**と判明する。彼女が"仕えてきた"と思っていた一族は、実は彼女自身が興したものだった
- 月夜家の「口伝せず、過酷な修行だけで技を継承する」という掟は、**円が周期的に記憶を失う自分自身のために、言葉ではなく体に技を刻み込むしかなかった**ことの名残りだったと示唆される
- 藩主暗殺の黒幕（重臣・**鴉丸(からすまる)**）は、円が「千年前から存在する当人」だと気づき、守ろうとした藩主を殺害し、円に濡れ衣を着せた守人の手先という位置づけは維持する。単純な悪役として設計（権力欲と守人への忠誠心のみ、事情や後悔は持たせない）

### 1.2 外伝『欠けた円環、満ちるとき』

1. 東国、月夜家の本拠地へ帰還。血縁も、親しいと思っていた人々も、すべて千年前の関係者の子孫でしかないという事実に直面する
2. 一族の記録を辿り、円自身が月夜家の初代当主だったことが判明する
3. **さらに深い真実が明かされる**: 本来ネフェスに対抗できる力を持つのは円自身だったが、身代わりとなった双子の姉・**環(たまき)**が犠牲になった。ネフェスはこれを勘違いし、**環の魂の方を100年ごとに生まれ変わらせ続けてしまっている**（円が999年眠っていた間、ネフェスは本物の円の存在に気づいていない）
4. 千年の間、守人はネフェスの指示のもと、**100年ごとに生まれ変わる環の魂を持つ者を狩り続けてきた**（円の一族に関わる特殊な例外として、世界全体の生贄構造とは別枠で扱う）
5. 現在、**1000年目（10回目）の生まれ変わりとして、環の魂が現代のある子ども・珠(たま)に宿っている**。守人はネフェスの指示でこの子を狙っている
6. クライマックス: **珠を守るため、守人（鴉丸）との対決**になる
7. 決着後、珠は保護され、安全な場所に預けられて**物語から退場する**（パーティの5人目にはならない）
8. 円は、自分が眠っていた999年の間ずっと、環が自分の代わりに苦しみ続けてきたという事実の重さと向き合う。ここでの決意が、終盤のネフェス本体との対峙に直接繋がる

## 2. 聖王国の仲間 — ヴァルド・ストランド(Vard Strand)

- **名前**: ヴァルド(Vard) — 北欧圏で「守護者」を意味する名前
- **一族**: **ストランド家**。北欧の実在の姓で「浜辺・海岸」の意。貴族ではなく、**代々"聖王国の防衛"を担う下級騎士の家系**。華やかな武功より、地道に守り続けることを美徳とする実直な家風
- **属性**: 水（回復・防御特化、大盾を使う壁役）
- **表向きの物語**: 封印の儀の護衛任務中、恐怖から持ち場を離れて逃げ、育ての親代わりの司教を死なせてしまった元従騎士。以来、贖罪を抱えて生きている
- **裏の真実**: 司教もヴァルドの正体(千年前の魂の生まれ変わり)に気づいており、儀式を通じて彼を守ろうとしていた。守人の妨害により死亡

### 2.1 外伝『届かぬ想いと裏切り』

1. 聖王国ヴェルダン、封印の儀が行われた聖堂跡へ帰還
2. 序盤で見つけていた「司教が書きかけた一文の切れ端」の手がかりを辿り、あの夜の"襲撃者"の正体が守人の手先だったことが判明
3. 司教の死後、教会の権力の空白に付け込んで**司教代行**の地位に昇り詰めた人物こそ、**かつての同輩騎士ロデリック(Roderick)**であることが分かる
4. ロデリックは、**病に倒れた妻と子の治療費**のため、真相の隠蔽とヴァルドを"逃げた臆病者"に仕立てる守人への協力を受け入れていた——ヴァルドと鏡合わせの、やむを得ず道を踏み外した人物として描く
5. ボス戦（中ボス「沈黙の司教代行」＝ロデリック、水属性）で対峙
6. ロデリックは敗れ、**最後に後悔を口にして死ぬ**（悲劇的な決別。命乞いも寝返りもしない）
7. 司教が本当に書き残そうとしていた言葉が明らかになる: 司教はヴァルドの正体に気づいた上で、最後まで彼を守ろうとしていた
8. 決着として、ヴァルドは大盾を「ヘイトシュルド(誓いの盾)」と自ら名付け直し、ただの罪の象徴ではなく**"誓いの盾"として掲げ直す**

## 3. 大燕の仲間 — 司馬燿(しば よう)

- **名前**: 司馬燿(しば よう)。「司馬」は史実で晋を建てた名門の姓。「燿」は火を含み「輝く・きらめく」を意味する字
- **立場**: **大燕の皇子(皇帝の直系の子)**
- **父帝について**: 悪徳の皇帝で、民から過剰な税を取り立て、一部の民を奴隷同然に扱っていた。裏では**守人と結託し、その活動資金源**になっていた
- **属性**: 火（黒い炎によるデバフ・状態異常特化）
- **経緯**:
  1. 燿は民を救うため密かに反逆を企てるが、父帝に露見し拷問を受ける
  2. その後、燿が千年前の生贄の魂の生まれ変わりであることが発覚し、守人に引き渡されそうになる
  3. **幼馴染の平民・林誠(りん せい)が、命と引き換えに燿を逃がした**
  4. **妹・司馬雪(しば せつ)は宮廷に残されたまま**。父の圧力下で燿を"謀反人"と信じ込まされている可能性があり、今後(外伝)で再会・真実との対峙が描かれる余地を残す

### 3.1 外伝『不知の阿妹、弑逆の刻』

1. 大燕へ帰還。林誠が命を落とした場所、あるいは幽閉されていた場所から物語が始まる
2. 父帝と守人の結託を示す証拠を集めながら宮廷内を進む
3. 妹・司馬雪との再会。**雪は最初、燿を謀反人として反発するが、真実(父帝の圧政と守人との結託)を知り、兵を率いて燿の味方につく**
4. 中ボス「粛清官（帝の密偵）」との対決。父帝の意を受け燿を追っていた人物で、宮廷・父帝への道を阻む最後の壁として配置する
5. **父帝との直接対決**。戦いの末、燿は父帝を討つ（命を奪う結末）
6. 暴政が終わり、大燕は守人への資金提供という後ろ盾を失う。妹・雪が後の統治を担う可能性を示唆して締める（詳細は今後）

## 4. 封守の民の仲間 — サユ

- **名前**: サユ(変更なし)
- **一族**: **封守の民(ふうしゅのたみ)**。「封じ、守る」の意
- **一族の特別な力**: 
  1. **魂を感じ取る力** — 生まれ変わった魂の気配を聴き分けることができる
  2. **「勇者の欠片(ゆうしゃのかけら)」を代々守ってきた一族** — 主人公が光属性に覚醒するための鍵となるアイテム。**封守の民の血を引く者にしか開けられない封印**がかけられている
- **守人に狙われた理由**: 魂を感知する力と、覚醒アイテムを封じる封印の力、その両方を封じるため一族ごと狙われた
- **属性**: 風（精霊・踊りによる味方バフ特化）
- **サユ自身の立場**: 部族壊滅前は、特別な立場ではないごく普通の子供だった
- **部族壊滅の経緯**: **モンスターと人間の両方**による襲撃を受ける。両親が戦って敵を食い止めている間、**祖母がサユだけを連れて逃げようとした**。しかし逃走の途中で敵に追いつかれ、**祖母がサユだけを逃がして自らが犠牲になった**。サユは、両親だけでなく祖母までも自分のために犠牲になったという事実を背負ったまま、唯一の生存者になる

### 4.1 外伝『血脈を封じ、祖母は殉ず』

1. 封守の民の故郷（壊滅した集落跡）へ帰還。祖母や両親の遺品・痕跡と向き合う
2. 襲撃の真相が判明: **交易商人が、一族の居場所と弱点を金と引き換えに守人へ売った**ことが発端だった
3. サユは封守の民の血を引く者として、**「勇者の欠片」の封印を解き**、ソティリスに手渡そうとする
4. まさにそのタイミングで交易商人（守人の手先）が現れ、欠片を奪おうと妨害してくる。**この戦いがそのままボス戦となり、戦闘の中でソティリスが光属性へ覚醒する**（§5参照。この外伝が覚醒シーンそのものを兼ねる）
5. 交易商人は敗れても**一切の後悔を見せず、命乞いもせず、「ただの商売だった」とだけ言い残して死ぬ**。ロデリックの悲劇的な決別とは対照的な、救いのない後味を意図的に残す
6. サユの癒やしは商人の死からではなく、**祖母や両親の想いを自分の意志で引き継ぐ**ことからもたらされる。封印を解いたのは義務としてではなく、自らの選択として描く

## 5. ソティリスの覚醒タイミング【確定】

サユ編（封守の民、故郷への帰還）にて、封守の民の血を引くサユが封印を解き、**「勇者の欠片」を主人公ソティリスに手渡す**。その直後にボス戦へ突入し、**戦闘の中でソティリスが光属性へ覚醒する**。

これにより、サユの外伝は単なる過去の清算ではなく、**パーティ全員が揃い、主人公が真の力に目覚める物語全体の転換点**を兼ねることになる。
