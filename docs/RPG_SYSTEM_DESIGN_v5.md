## 1.1 戦闘コマンド【新設】

ドラゴンクエスト式のコマンド体系を採用する。

| コマンド | 内容 |
|---|---|
| こうげき | 通常攻撃。**SPを一定量回復する** |
| スキル | 設計した技を使用する。**すべてのスキルがSPを消費する**（キャラごとの個別スキルにSP消費量を都度明記しない場合も、この共通ルールが適用される） |
| どうぐ | 回復アイテム、状態異常回復アイテムなどを使用する |

**「にげる」は存在しない**。代わりに画面右上に「やめる」ボタンを配置し、押すと戦闘が終了する。次に戦闘を始める際は、その戦闘の最初からやり直しになる（中断・再開の概念はない）。

## 3. 属性システム

属性は火・水・風・光・闇・無属性の6種（変更なし）。

| 属性 | 該当キャラ | イメージ |
|---|---|---|
| 無属性 | ソティリス（覚醒前） | 全属性に対して弱点/耐性なし（すべて通常倍率） |
| 光 | ソティリス（覚醒後） | 下記「覚醒システム」参照 |
| 闇 | 円 | 空間転移による接近戦、切り伏せる攻撃 |
| 水 | ヴァルド | 防御・回復、贖罪のイメージ |
| 火 | 司馬燿 | 黒い炎によるデバフ・状態異常 |
| 風 | サユ | 精霊・踊りによる支援 |

**耐性倍率の基準**: 弱点=×1.5、通常=×1.0、耐性=×0.5、無効=×0

### 3.1 火・水・風の三竦み【新設】
火・水・風の3属性は、三竦みの相性を持つ。無属性・光・闇はこの三竦みには含まれない。

| 攻撃属性＼防御属性 | 火 | 水 | 風 |
|---|---|---|---|
| **火** | 通常(×1.0) | 耐性(×0.5) | 弱点(×1.5) |
| **水** | 弱点(×1.5) | 通常(×1.0) | 耐性(×0.5) |
| **風** | 耐性(×0.5) | 弱点(×1.5) | 通常(×1.0) |

- 火は風に強く(弱点を突ける)、水に弱い(耐性を持たれる)、**同じ火属性同士は通常倍率**
- 水は火に強く、風に弱い、**同じ水属性同士は通常倍率**
- 風は水に強く、火に弱い、**同じ風属性同士は通常倍率**

この相性は、司馬燿(火)・ヴァルド(水)・サユ(風)の3人の間で自然に得意不得意が生まれる設計。また敵の属性(§5参照)ともこの三竦みで絡む。

### 3.2 光・闇の特殊能力【新設】
光と闇は三竦みに参加しない代わりに、それぞれ固有の「防御無視」特殊能力を持つ。

- **光**: 攻撃時、相手の**属性防御(属防)を無視**する。属防がどれだけ高い相手にも、その値を計算に入れずダメージを通す
- **闇**: 攻撃時、相手の**防御力(DEF)を無視**する。DEFがどれだけ高い相手にも、その値を計算に入れずダメージを通す

この2つは弱点/耐性倍率とは別枠の「防御貫通」効果として扱う（弱点倍率と防御無視は重複して適用される）。

- ソティリス(覚醒後・光)は、既に「常に弱点倍率」を持つ上に属防も無視するため、属性面で最強クラスの攻撃力になる
- 円(闇)は、物理攻撃(太刀)がベースのため、DEF無視との相性が特に良い。属防が低い(20)代わりに、DEFの高い敵に対しても安定して火力を出せる設計になる

### 3.3 覚醒システム（ソティリス専用・確定版）
- **攻撃面**:
  - 覚醒後、ソティリスの与えるダメージは相手の属性に関わらず**常に弱点倍率(×1.5)扱い**
  - **全ステータス(HP/SP/ATK/DEF/属攻/属防/SPD)に一律+100**
- **防御面**:
  - 覚醒後は一切の耐性を持たなくなり、あらゆる属性の攻撃を**常に×1.5で受ける**
- 攻撃力・全ステータスが跳ね上がる代わりに、被ダメージも跳ね上がるハイリスク・ハイリターン仕様

## 4. パーティメンバーのステータス・武器（Lv1基準）

### ソティリス（無属性→覚醒後：光属性）
- HP50 / SP50 / ATK50 / DEF50 / 属攻50 / 属防50 / SPD50 / CRT率20% / CRTダメ120%
- **覚醒後(参考値・Lv1時点)**: HP150 / SP150 / ATK150 / DEF150 / 属攻150 / 属防150 / SPD150（全ステータス+100）
- **武器**: 剣「**エオス(Eos)**」— ギリシャ神話の曙の女神の名。無属性の間はただの剣だが、光属性への覚醒を静かに予告する名前
- **スキル**:
  - ラピッドスラッシュ: 単体攻撃
  - ヒーリング: 仲間を1人回復
  - ソウルブレイク: SP消費の属性攻撃

  - ラピッドスラッシュ → **ルミナスラッシュ**: 光を纏った一撃。属防無視の特性がそのまま乗る
  - ヒーリング → **ライフドレイン**: 敵からHPを吸い取り、**吸収したHP量に応じてパーティ全体を回復**する。自己吸収ではなく支援として機能する
  - ソウルブレイク → **ソウルバースト**: より大きな爆発力を持つ属性攻撃
  - 【必殺】エオスバースト → **エオス・レイディアンス**: 曙が完全に昇りきった、という意味を込めた最終形態の必殺技
- 役割: バランス型。初期性能は強すぎず弱すぎない基準値として設計。無属性時は安定した汎用性、光属性覚醒後は攻撃面で圧倒的な火力を発揮する代わりに防御が崩壊するハイリスク・ハイリターン型

### 4.1 武器・装備システム【新設】

#### 4.1.1 武器システムの構造
各キャラクターは固有武器（エオス、黄泉比良坂、ヘイトシュルド、玄火、継風）を1本持ち、物語進行に応じて強化・進化する。加えて、宝物（遺物）による補強を組み合わせるハイブリッド型とする。

#### 4.1.2 固有武器の進化段階
4段階。各章クリアごとに強化される。属攻・属防・SPD・CRTダメはレベルアップでは伸びない設計（§2参照）のため、武器進化がこれらを伸ばす主な手段になる。

| 段階 | 条件 | 属攻 | 属防 | SPD | CRTダメ |
|---|---|---|---|---|---|
| 第1形態 | 初期 | +0 | +0 | +0 | +0% |
| 第2形態 | 1章クリア後 | +20 | +20 | +10 | +15% |
| 第3形態 | 2章クリア後 | +45 | +45 | +20 | +30% |
| 第4形態 | 3章クリア後 | +80 | +80 | +35 | +50% |

※上記は全キャラ共通の目安値（叩き台）。キャラごとの役割に応じた配分差は今後の検討課題

#### 4.1.3 宝物（遺物）の装備スロット
**3枠**。`RELICS.md`の宝物リストから選択して装備する。

- **放置中（オート進行）の敵は一撃で倒せる仕様**とし、この間は雑魚敵のステータスは実質意味を持たない
- **雑魚敵は単独では出現せず、ボス戦に同伴する形でのみ登場する**（マニュアル操作の戦闘でのみステータスが意味を持つ）

### 5.3 スキル倍率の階層【新設】

中ボスの基礎ステータス（上表）はそのまま「通常攻撃」の基準値として維持し、スキル使用時はここに倍率を乗算する。

| 技の格 | 倍率 | 結果（味方の耐久発数換算） | 使用頻度イメージ |
|---|---|---|---|
| 通常攻撃 | ×1.0 | 約7〜8発 | 毎ターン |
| 強攻撃 | ×1.5 | 約5発 | 数ターンに1回 |
| 大技（必殺級） | ×2.0〜2.5 | 約3〜4発 | 戦闘中数回。予兆演出とセットにし、プレイヤーが防御・回復で対応できる猶予を作る |
