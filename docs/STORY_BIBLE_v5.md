## 2. アナスタシスの行商人「アナスタス」【確定】

### 2.1 立場と正体
- 表向き: ショップに現れる行商人。掘削道具の強化・地質調査などを提供する、便利だが一見脇役のNPC
- 裏の正体: **アナスタシスの現指導者**。商売を通じてパーティの発掘状況（文書の発見数、遺物の入手状況など）を陰から把握し、守人に見つかって消される前に、パーティが自分たちの本当の力に気づけるよう誘導する役割を担う

### 2.2 名前
**アナスタス(Anastas)**。組織名「アナスタシス(Anastasis＝復活)」と同じ語根を持つ、実在するギリシャ語圏の人名。序盤から繰り返し目にする名前でありながら、答えそのものを最初から晒しているという仕掛け（ソティリス/ソストラトスの語根共有と同じ設計思想）。

### 2.3 決まり文句とその変化【確定】
- 通常時、ショップ訪問のたびに必ず一言: **「良い掘り出し物を!」**
- **正体解禁シーンの直前1回のショップ訪問のみ**、この決まり文句が **「また会おう」** に変化する。演出上の強調は一切行わない（BGM変化や表情差分などのUI的な予告をしない）。セリフの違いに気づけるかどうかだけの、地味な予告にする
- この変化は物語を通じて**1回のみ**。他の場面では発生しない

### 2.4 導線設計【確定】
各章の節目（本編を先に進める前）には、**必ずショップで何かを購入しないと物語を進行できない**ゲート状のイベントを設ける。これにより、プレイヤーがショップを素通りして§2.3の仕掛けを見逃す事故を防ぐ。

## 3. ソストラトスの名の初出エピソード（序章・改訂版）【確定】

v3/v4にあった「本筋から外れた行き止まりの横穴で偶然発見する古いつるはしの柄」案を廃案とし、以下に置き換える。

### 3.1 配置
**序章、レトスが初めて主人公につるはしを渡す場面**に統合する。チュートリアルもろくに始まっていない、物語の一番最初の段階に置く。プレイヤーの意識がまだ薄いうちに、答えを最初から渡してしまうことを狙う。

### 3.2 やり取り（ドラフト）
1. レトスが「もう掘れる歳だ」といった一言と共に、自分の使い古したつるはしを息子に譲る
2. 主人公はそれを調べ、柄に刻まれた「ソストラトス」の文字に気づく。何気なく「これ、ソストラトスって彫ってあるけど?」と尋ねる
3. レトスは「・・・・・・・・」と黙り込んだ後、「さあな……ずっと前からそうだった」とだけ返す
4. これ以降、この話題は二度と登場しない

### 3.3 演出上の禁止事項
発見演出・実績通知・UIの強調表示・BGM変化は一切なし。プレイヤーが「なんとなく気になる一言だな」程度にしか感じない温度で流す。

### 3.4 チュートリアルUI(メタ層)との関係（v3/v4から変更なし）
チュートリアルの案内文は、ゲーム開始時から一貫して「ソストラトス」の名で署名される、作中人物には見えないプレイヤー向けの情報。§3のエピソードとは独立した層として併存し、両者が同一人物を指すとは、正体解禁シーンまでプレイヤーには気づかせない。

### 3.5 このつるはしとアナスタシスの意匠との関係【確定・重要な区別】
仲間4人の"小さなエピソード"アイテム（v4§5）には、アナスタシスの象徴「欠けた円」を共通して刻む設計にしたが、**このつるはしの柄には、その意匠を入れない**。ソストラトスはアナスタシスと無関係な一般人（v4§2.3で確定済み）であるため、意匠を入れると設定の一貫性が崩れる。区別を保つこと。
