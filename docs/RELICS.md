## 現在の宝物リスト

| 名称 | 説明文 | 効果 |
|---|---|---|
| **アビュソスの欠片** | 大穴の縁で見つかる黒い結晶。装備者の魂力を高めるが、装備するたびに小さな眩暈を伴う。 | 属攻+15 |
| **レテの雫** | 涙の形をした透明な石。触れると一瞬、誰かの記憶の断片が流れ込む。 | 属防+10、被弾時ごく低確率で敵の弱点を看破 |
| **五弁の首飾り** | 5枚の花弁を象った古い装身具。誰が作ったのかは分からない。 | パーティ全員のCRT率+3% |
| **黄泉比良坂の鞘** | 円の刀に合う、行方不明だったはずの鞘。 | 円装備時、被ダメージ-10% |
| **贖罪の紋章** | ヴァルドの実家に伝わるはずだった、授与されなかった勲章。 | ヴァルド装備時、回復効果+15% |
| **封じられた禁書の頁** | 司馬燿が持ち出した禁書の、燃え残った一頁。 | 状態異常付与率+10% |
| **風読みの羽根** | サユの部族に伝わる、風を読むための儀式道具の欠片。 | SPD+8、逃走成功率上昇 |
| **刈り取られし者の骨片** | ネフェスの眷属の骨。持つと僅かに悪寒がする。 | ATK+10、HPが低いほど追加ダメージ |
| **アナスタスの銅貨** | 行商人が「おまけだ」と渡してくる、変哲のない銅貨。売ろうとしても、なぜか誰も買い取ってくれない。 | CRTダメ+10%（売却不可） |
| **欠けた円環** | 半分だけの、金属製の輪。もう半分はどこかにあるようだ。4つ集めると、何かが起きるらしい。 | パーティ全員の属防+8（4個収集で追加効果） |
