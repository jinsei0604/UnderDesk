## 1. 世界観（変更なし・要約）

世界の名は「オリガ」。4つの文化圏（東国＝和風／聖王国ヴェルダン＝西洋風／大燕＝中華風／高嶺の民＝山岳民族風）は、大陸中央の大穴によって隔てられている。大穴の最深部には「刈り取る者（ネフェス）」が眠り、罪と記憶を喰らう。

**v2での重要な確定事項**: ネフェスは実在の脅威であると同時に、**1000年前に4つの文化圏が共同で行った「たった1つの出来事」の記憶そのもの**として設計する。以降のすべての伏線は、この「1つの出来事」を5つの異なる断面（主人公＋仲間4人）から見せる構造で統一する。

## 2. 核心の真実（最終開示情報・ネタバレ厳重管理）

> この節はプレイヤーに直接開示しない。Claude Codeと執筆者だけが参照する「正解」。以下から逆算して伏線を配置する。

- 1000年前、4つの文化圏の代表者たちは、大穴から溢れ出した大災厄（詳細は§2.1）を鎮めるため、**5人の子供**（各文化圏から1人ずつ＋「境界に生まれた」子供1人の計5人）を大穴に捧げた。
- 5人のうち4人は文化圏の代表として選ばれた子供、残る1人は**どの文化圏にも属さない、大穴の縁で生まれた名もなき子**だった。この5人目こそが現在の**主人公の前世／魂の残響**である。
- 捧げられた5人の意識と記憶が混ざり合い、罪悪感の集合体として生まれたのがネフェス。
- 4つの文化圏はそれぞれ、自分たちが子供を「捧げた」側の記憶だけを都合よく書き換え、「守るために封印した」「悪しきものに奪われた」という伝承に変えて後世に伝えた。**4つの伝承はすべて同じ事件の異なる嘘**である。
- 千年の間、4つの文化圏の中枢（東国の穴守りの家系／聖王国の教会上層部／大燕の宗門／高嶺の民の長老会議）は、この真実を代々の最高責任者だけに口伝してきた。**「守人（もりびと）」**と呼ばれるこの秘密の継承者たちが、真実を隠すために暗躍する存在として全編に薄く配置される。
- 仲間4人の個人的な悲劇は、**すべて「守人」の暗躍によって引き起こされた**（偶然ではない）。彼らが冤罪を着せられ、実験体にされ、儀式を失敗させられたのは、**真実に近づきすぎたから**、あるいは**千年前の生贄の血を引く家系だから**。
- 主人公自身が幼少期に失ったもの（v1では未確定としていた）は、**「5人目の子」としての前世の記憶そのもの**。大穴に落ちたことでその残響が疼き、無意識に掘り進んでいる。

### 2.1 大災厄の正体（推奨案）
「地の底から漏れ出した瘴気が万物の記憶を溶かし、触れた者から人格を奪っていく」現象（＝ネフェスの萌芽そのもの）とする。**病でも侵略でもなく、「忘却」自体が災厄**という設計にすると、ネフェスの能力（記憶を喰らう）と起源が完全に一致し、伏線の整合性が最大化する。

## 5. 執筆・実装への落とし込み方針

### 5.1 データ構造への反映（JSON/CSVのみで完結）
既存の文書データ形式に、以下のフィールド追加を推奨する（コード変更不要、`data/`側の追記のみ）:

```json
{
  "id": "doc_riko_003",
  "companion_tag": "riko",
  "strata": 1,
  "foreshadow_ids": ["F01", "F04"],
  "reveal_stage": "surface",
  "text_key": "doc_riko_003"
}
```

- `reveal_stage`: `surface`（表の物語段階）／`mid`（半開示）／`payoff`（回収）の3値。CLAUDE.mdの「文書出土条件」ロジックにそのまま乗せられる。
- 伏線カタログ（本資料§4）のIDを台帳として、**各文書がどの伏線をどの段階で担うかを一覧管理できるようにする**。これにより「進撃の巨人レベル」の物量になっても矛盾を防げる。

### 5.2 執筆優先順位（CLAUDE.mdのブロッカー解消のため）
1. **最優先**: リコの文書1〜5編（加入まで）を、F01・F04・F02の"surface"段階だけを使って先に完成させる（現在の実装を即座に埋められる）
2. リコ外伝（東国、`payoff`段階の一部＋F05・F09の"mid"）
3. 以降、§3.1のマイルストーン順（アルバン→青嵐→サユ）で同様に執筆
4. 最終章の伏線回収文書（F01〜F12すべての"payoff"）はP4以降、全外伝執筆後に着手（矛盾が出た場合の手戻りを最小化するため）

### 5.3 マーケティングへの接続
- 「全ては繋がっている」を訴求する場合、**発売前の情報公開でも伏線の存在自体は隠さず、"何が繋がっているか"だけを隠す**戦略が有効（進撃の巨人も"半分明かして煽る"手法で話題化した）。
- 施策案: コレクション画面のスクリーンショット（F10の4印章が並ぶ場面など）を、答えを言わずにSNSで公開し考察を誘発する。
- 全伏線回収済みの「真エンディング」到達者だけが見られる特別な調査書カード（デイリーシステムと連動）を用意し、SNS上で「自分は繋げられた」という達成感を共有させる。
