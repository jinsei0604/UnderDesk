# 物語設定資料集 v6『UNDERDESK 〜忘却の底で〜』
## 統合版（v4＋v5）

> v4（神話構造の全面改訂）とv5（世界名・タイトル副題・アナスタス・ソストラトスの序章エピソード）を1つに統合した最新版。

---

## 1. タイトル・世界名

- **タイトル**: UNDERDESK 〜忘却の底で〜
- **世界名**: アビュソス(Abyssos) — ギリシャ語で「深淵」。物語の中心にある大穴そのものを指す言葉

---

## 2. 世界観

世界アビュソスには、4つの文化圏が存在する。

| 地方 | 文化モチーフ |
|---|---|
| 東国（あづまのくに） | 和風・武家社会 |
| 聖王国ヴェルダン | 西洋中世・騎士と教会 |
| 大燕（たいえん） | 中華風・科挙官僚制 |
| 封守の民（ふうしゅのたみ） | 山岳民族・精霊信仰 |

大陸の中心には、千年前に開いたまま塞がらない巨大な穴がある。最深部には「刈り取る者（ネフェス）」が眠っている。

---

## 3. 核心の真実

- 千年前、**ネフェスは4つの国を滅ぼそうとしていた**。これを止めるため、4つの文化圏の代表者たちは5人の子供（各文化圏1人＋「境界に生まれた」名もなき子1人）を大穴に捧げた
- この生贄によってネフェスは滅びず、**封印**された
- 千年の間、封印は少しずつ緩み続けてきた。**現在、ネフェスは再び目覚めかけている**。これに呼応するように、千年前に捧げられた5つの魂が、主人公たちの世代に生まれ変わって現れ始めた——これが「なぜ今、この5人なのか」の答えになる
- 4つの文化圏はそれぞれ、自分たちが子を「捧げた」側の記憶を都合よく書き換え、「守るために封印した」という伝承に変えた

---

## 4. 二つの思想— 守人とアナスタシス

- **守人（もりびと）**: 千年の間、4つの文化圏の中枢に根を張ってきた秘密の継承者たち。「かつてと同じように、5つの魂を再び生贄に捧げれば、ネフェスをもう一度封印できる」と信じ、今回生まれ変わった5人を狙っている。仲間4人それぞれの悲劇は、すべてこの守人の暗躍が原因
- **アナスタシス(Anastasis＝復活)**: 千年前に子を捧げた親たちが、「本当は、あの子たちの力でネフェスを完全に滅ぼせたのではないか」という可能性に気づき結成した組織。**封印ではなく、完全な討滅**を目指す。千年の間、守人の弾圧でほぼ壊滅寸前に追い込まれながらも、各文化圏に潜伏し続けてきた

物語は、「主人公たちは"また生贄にされる側"なのか、"今度こそ運命を変える側"なのか」という問いを軸に進む。

### 4.1 アナスタスという行商人
アナスタシスの現指導者は、ショップに現れる**行商人アナスタス(Anastas)**。組織名「アナスタシス」と同じ語根を持つ人名を、正体を隠さぬまま最初から使い続けている。表向きはただの脇役NPCだが、商売を通じてパーティの発掘状況を陰から把握し、守人に消される前に本当の力へ気づけるよう誘導している。

- 通常時の決まり文句: 「**良い掘り出し物を!**」
- **正体解禁シーンの直前1回のショップ訪問だけ**、「**また会おう**」に変化する（演出強調なし）
- 各章の節目に進む前は、必ずショップでの購入を挟む導線にして、この仕掛けを見逃しにくくする

---

## 5. 主人公 — ソティリス(Sotiris)

- 名前の意味: ギリシャ語「sozo(救う)」由来の「救世主」。東国育ちなのに東国らしくない名を持つのは、**「どの文化圏にも属さない5人目の魂」の生まれ変わり**であることの伏線
- **掘り進む動機**: 恩人レトスが命がけで探し続けている「何か」を、代わりに見つけてやりたいという個人的な願い。この時点でレトスが実の父だとは知らない

---

## 6. 父 — レトス(Lethos) / 本名ソストラトス(Sostratos)

### 6.1 プロフィール
- 現在の名: レトス（ギリシャ神話「忘却の河・レテ」が由来）
- 本当の名: ソストラトス（「sozo(救う)」の語根をソティリスと共有。意味は「軍勢/一家を救う者」——息子が"世界"を救う名を持つのに対し、父は"自分の家族だけ"を救おうとした男、という対比）
- アナスタシスとは無関係な一般人

### 6.2 過去の真実（時系列）
1. 妻（主人公の母）が守人に洗脳され、今回生まれ変わった「5つの魂」の1つとして息子を大穴に捧げようとする
2. ソストラトスはそれを止めるため、母を殺すしかなかった。「父親としての資格はない」と感じながらも、息子だけは今まで通り生かしたいと育て続けようとする
3. しかしすでに母は、5つの魂を持つ者専用に調整された特別な毒を息子に盛っていた
4. ソストラトスはネフェスのもとを訪れ、息子の毒を治すよう懇願。「父親としての記憶」と「母親殺しの罪の記憶」を差し出す条件を了承する
5. ネフェスは記憶を奪うだけでなく改ざんする。ソストラトスの記憶は「大穴に繋がる洞窟を掘り進め、ある宝物を探している」という強迫観念に書き換えられる。**その"宝物"の正体は、彼自身が失った記憶そのもの**
6. さらにネフェスは、息子の毒をソストラトス自身に移し替える。この毒は5つの魂を持つ者には作用が弱いが、通常の人間には致死性を持つ。**息子が大切な人を失う悲しみを理解できる年齢に達し、一番悲しみが大きくなる日に死ぬよう調整して**移植された
7. 死の間際、レトスはすべての記憶（父親であったこと、母を手にかけた罪、本当の名）を取り戻す。それまでは、主人公を「息子」だと認識していなかった

### 6.3 序章のエピソード（ソストラトスの名の初出）
物語の一番最初、レトスが「もう掘れる歳だ」と、自分の使い古したつるはしを息子に譲る場面に組み込む。

1. 主人公は柄に刻まれた「ソストラトス」の文字に気づき、何気なく「これ、ソストラトスって彫ってあるけど?」と尋ねる
2. レトスは「・・・・・・・・」と黙り込んだ後、「さあな……ずっと前からそうだった」とだけ返す
3. これ以降、この話題は二度と登場しない
4. 発見演出・実績通知・UI強調は一切なし

これとは独立して、**チュートリアルUI（プレイヤーにしか見えないメタ層）は、ゲーム開始時から一貫して「ソストラトス」の名で署名される**。プレイヤーは「いつも助けてくれる、名前だけ知っている人」として自然に愛着を持つ。この2つの層が同一人物を指すとは、正体解禁シーンまで気づかせない。

※このつるはしには、アナスタシスの象徴（後述、欠けた円）を入れない。ソストラトスはアナスタシスと無関係な一般人であるため。

## 7. 仲間4人

100年周期の生贄設定は廃止したため、4人が狙われる理由は「ネフェスの目覚めに呼応して、千年前の5つの魂の1つとして生まれ変わったから」に統一されている。

| 仲間 | 表向きの物語 | 裏の真実 |
|---|---|---|
| 円（東国） | 藩主暗殺の濡れ衣で追われる記憶喪失の武士。元・東国最高位の武士（通り名: 黄泉） | 藩主は円が千年前の魂の生まれ変わりだと気づき、守ろうとして守人に消された。円ごと消すため濡れ衣を着せられた |
| ヴァルド（聖王国） | 封印の儀を放棄して司教を死なせた臆病な元従騎士。実家は貴族ではなく、代々国防を担う下級騎士の家系 | 司教もヴァルドの正体に気づき、儀式を通じて彼を守ろうとしていた。守人の妨害により死亡。かつての同輩騎士ロデリックは、病の妻子の治療費のため守人に協力し、司教代行の地位を得ていた |
| 司馬燿（大燕） | 暴君の父帝への反逆に失敗し拷問を受けた皇子。幼馴染・林誠の犠牲で逃れる | 司馬燿自身が魂の生まれ変わり。父帝は守人と結託し、その資金源だった。妹・司馬雪は宮廷に残されたまま |
| サユ（封守の民） | 部族壊滅の唯一の生存者。両親が敵を食い止める間、祖母に連れられ逃げるが、祖母が身代わりとなり犠牲になった | 封守の民は魂を感じ取る力と、主人公の覚醒の鍵「勇者の欠片」を代々守ってきた一族。その両方を封じるため、モンスターと人間の両方による襲撃で守人に一族ごと狙われた |

※各仲間のストーリー詳細（外伝の展開など）は今後のセッションで設計する（未着手）。

### 7.1 仲間4人の小さなエピソード
父のつるはしと対になる仕掛けとして、仲間4人にもそれぞれ見逃しやすいアイテムを1つずつ配置する。

| 仲間 | アイテム | 中身・扱い |
|---|---|---|
| 円（東国） | 古い石 | 岩壁の一部に、円の本当の家系の紋が薄く彫られている |
| ヴァルド（聖王国） | 紙の切れ端 | 教会の瓦礫に紛れた、書きかけの一文 |
| 司馬燿（大燕） | 名もない装飾品 | 妹（司馬雪）の私物らしき小さな飾り |
| サユ（封守の民） | 古い木片 | 部族の唄の最後の一節だけが刻まれている（翻訳表示なし） |

4アイテムすべてに、アナスタシスの象徴「**欠けた円が再びつながる**」意匠をごく小さく共通して刻む。発見演出・実績通知は一切なし。
